How it works
Data is fetched by a Python script (fetch_data.py) running on GitHub Actions every weekday at 12:00 IST. It downloads 1-year daily prices via yfinance, computes log-return correlations for all Nifty 500 stocks, and saves the result to data/corr_data.json in this repo. This dashboard loads that file — no live API calls.
